Hepsisurada

React, Flask (Python), MongoDB

September 2024 - February 2025

Hepsisurada is a fully functional online shopping platform developed by a team of six. The system includes both a customer facing interface and a dedicated internal panel for staff such as product managers.

The frontend was developed using React, and the entire interface, including both user and staff views, was implemented by me as the Frontend Lead. I was also responsible for the integration between the frontend and backend, ensuring secure and efficient communication across the platform.

The backend was built with Flask (Python) and the database was structured with MongoDB, managing users, products, and orders. The project demonstrates a full-stack architecture with role based access control, secure data handling, and a seamless user experience.
